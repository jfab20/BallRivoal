# The Ball Rivoal theorem

This is an attempt to formalize the Ball Rivoal Theorem in Lean4, which states that there are infinitely many $k \in \mathbb{N}$ such that $\zeta(2 k + 1)$ is irrational.

We use the proof by Ball and Rivoal in their article "Irrationalité d’une infinité de valeurs de la fonction zêta aux entiers impairs" which can be found [here](https://rivoal.perso.math.cnrs.fr/articles/zetaimp.pdf).

However, we present a more detailed explanation of the article in [here](pdfs/proof-english.pdf). You can also find a version written in french [here](pdfs/proof-french.pdf)
